1)Префиксный вариант ++i:

let i = 0;
while (++i < 5) alert( i ); // 1, 2, 3, 4 -в сравнении и в alert используется  увеличенное на 1 значение i(префиксный инкремент)

2)Постфиксный вариант i++

let i = 0;
while (i++ < 5) alert( i ); //  1, 2, 3, 4, 5 -в сравнении используется "старое" значение i, а в alert попадает увеличенное на 1(постфиксный инкремент)
